CST-120 Web Development Syllabus

Course Description

This course introduces the fundamental concepts and syntax of the web development languages including HTML, CSS, and JavaScript. The course focuses on the foundation required to build complex dynamic web applications Prerequisite: none.

Weeks 1 & 2

Topic 1 HTML

Description:

In this topic, students are introduced to the HTML markup language. They will learn the basic structure, the basic elements of an HTML document, and how to work with text and basic controls. The students finish this topic by learning the developer tools available to them, as well as how web servers work to help support development of HTML based applications

Readings

  • W3Schools: Go to https://www.w3schools.com. Select the Learn HTML menu under the HTML and CSS section. Complete all reading and tutorials under the HTML Home Tutorial section.

Objectives

  1. Develop web pages with basic elements and structure that include, but are not limited to, document, head, and body tags.
  2. Develop web pages with text that include, but are not limited to, headings, paragraphs, formatting, fonts, and color tags.
  3. Develop web pages with components that include, but is not limited to, links, images, and lists tags.
  4. Investigate how a web server functions and identify differences between localhost, file URL, and domain names in the URL.
  5. Investigate the developer tools available as a web page developer.

Weeks 3 & 4

Topic 2 Advanced HTML

Description:

In this topic, students continue to expand their knowledge of the HTML markup language, learning some of the more advanced features as they discover how to work with layouts, graphics, and media content. The students finish this topic by learning how to make HTML pages that are ADA compliant.

Readings

  • W3Schools: Go to https://www.w3schools.com. Select the Learn HTML menu under the HTML and CSS section. Complete all reading and tutorials under the HTML Forms, HTML Graphics, HTML Media, and HTML API's Tutorial sections.

Objectives:

  1. Develop web pages with layouts that include, but are not limited to, Blocks/Div/Span, tables, and layouts tags.
  2. Develop web pages with graphics that include, but are not limited to, Image, Canvas, and SVG tags.
  3. Develop web pages with media that include, but are not limited to, video and audio tags.
  4. Explain how and why ADA compliance is achieved with ADA guidelines

Weeks 5 & 6

Topic 3 CSS

Description:

In this topic, students learn Cascading Style Sheets and how to apply styling to HTML pages using fonts, colors, and formatting. The students finish the topic by learning how to apply backgrounds, transitions, and animations to their HTML pages.

Readings

  • W3Schools: Go to https://www.w3schools.com. Select the Learn CSS menu under the HTML and CSS section. Complete all reading and tutorials under the CSS Tutorial and CSS Advanced Tutorial sections.

Objectives:

  1. Develop web pages with CSS Basics Styles that include, but are not limited to, Inline, Internal, External, and Selectors styles.
  2. Develop web pages with text that include, but are not limited to, built-in fonts, custom fonts from external sources, color, and alignment styles.
  3. Develop web pages with formatting and layouts that include, but are not limited to, backgrounds, borders, margins, padding, width/height, and the Box Model styles.
  4. Develop web pages that include, but are not limited to, transitions and animations styles.

Weeks 7 & 8

Topic 4 JavaScript

Description:

In this topic, students are introduced to the JavaScript programming language. This programming language is used as the foundation for client-side programming on web pages. Students learn the basic language syntax, how to work with variables, make decisions, perform repetitive operations, and create reusable functions

Readings

  • W3Schools: Go to https://www.w3schools.com. Select the Learn JavaScript menu under the JavaScript section. Complete all reading and tutorials under the JS Tutorial Tutorial section

Objectives:

  1. Develop code that integrates basic JavaScript using Inline statements in a web page
  2. Develop code that leverages variables and data types in a web page.
  3. Develop code that leverages operators (math, assignment, comparison, logical, boolean, and string) in a web page.
  4. Develop code that leverages arrays in a web page.
  5. Develop code that uses reusable functions in a web page.

Weeks 9 & 10

Topic 5 Advanced JavaScript

Description:

In this topic, students continue to expand their knowledge of the JavaScript programming language, learning how to create objects, work with data as arrays, and handle events from DOM objects from their HTML pages. Students finish the topic with an introduction to the JavaScript Object Notation (i.e., JSON), learning the advanced features of newer updates to the JavaScript programming language, and how this programming language is used alongside current web application frameworks that are used as the foundation to build modern, dynamic web applications.

Readings:

  • W3Schools: Go to https://www.w3schools.com. Select the Learn JavaScript menu under the JavaScript section. Complete all reading and tutorials under the JS Forms, JS Objects, JS Functions, and JS HTML DOM Tutorial sections.
  • W3Schools: Go to https://www.w3schools.com. Select the Learn JavaScript menu under the JavaScript section. Complete all reading and tutorials under the JS Browser BOM, JS AJAX, and JS JSON Tutorial sections

Objectives:

  1. Develop code that integrates basic JavaScript from external files in a web page.
  2. Develop code that uses decision and loop constructs in a web page.
  3. Develop code that leverages JavaScript objects in a web page.
  4. Develop code that handles events in a web page.
  5. Explain what JSON is and how this data format is used to build web applications.
  6. Investigate advanced features found in recent updates to the JavaScript programming language.
  7. Investigate recent JavaScript Web Application Frameworks that are available to build dynamic web applications.

Weeks 11, 12 & 13

Topic 6 Web Development Libraries

Description:

In this topic, students learn to use web development libraries to add dynamic, rich, and compelling content and user interfaces to their web pages. Students will also be introduced to the concept of responsive web pages, what this means, and how they can support this concept when building web pages.

Readings:

  • W3Schools: Go to https://www.w3schools.com. Select the Learn CSS menu under the HTML and CSS section. Complete all reading and tutorials under the CSS Responsive and CSS Grid Tutorial sections.
  • W3Schools: Go to https://www.w3schools.com. Select the Learn jQuery menu under the JavaScript section. Complete all reading and tutorials under the jQuery Tutorial, jQuery Effects, jQuery HTML, and jQuery AJAX Tutorial sections.
  • W3 Schools: Go to https://www.w3schools.com. Select the Learn Bootstrap menu under the HTML and CSS section. Complete all reading and tutorials for the latest version of Bootstrap under the Bootstrap Tutorial and Bootstrap Grid Tutorial sections.

Objectives:

  1. Develop web pages that support a responsive design.
  2. Develop web pages that leverage the use of jQuery, including, but not limited to, the use of Selectors, Events, DOM Manipulation, AJAX, and Data Table features.
  3. Develop web pages that leverage the use of Bootstrap, including, but not limited to, the use of the Grid System and various UI Components (Buttons, Text, Alerts, Tables, Forms, and NavBar) features.

Weeks 14 & 15

Topic 7 Advanced Web Development

In this topic students learn that building static web pages have their limits and that they often need to be deployed on web servers sometimes backed with a server-side programming language or database. This topic at a minimum will introduce the students to the HTTP protocol, client server model, HTML forms, and how to implement forms processing to future expand on the capabilities of the web pages they have built so far in previous topics. This topic introduces the students to the basic concepts required for future web application development courses where formal dynamic web application frameworks are used.

Readings

  • None.

Objectives:

  1. Investigate the HTTP protocol and client server model.
  2. Develop web pages with HTML Forms.
  3. Develop web pages that integrate Forms Processing on the server.
